## 📌 Overview

A clean, fast, and fully responsive personal portfolio website built without any heavy frameworks. Showcases my projects, skills, and journey as a frontend developer — designed to leave a strong first impression.

---

## ✨ Features

- 🎨 **Clean & Minimal Design** — Professional UI with smooth sections
- 📱 **Fully Responsive** — Works perfectly on mobile, tablet, and desktop
- ⚡ **No Framework Overhead** — Pure HTML, CSS & Vanilla JS for fast load times
- 🔗 **Project Showcase** — Live links and descriptions for all featured projects
- 📬 **Contact Section** — Easy way for visitors to reach out
- 🌙 **Smooth Scrolling** — Seamless navigation between sections

---

## 📂 Sections

| Section | Description |
|---------|-------------|
| Hero | Introduction and CTA buttons |
| About | Background and skill summary |
| Skills | Technologies and tools I work with |
| Projects | Featured projects with live links |
| Contact | Ways to connect |

## 🛠️ Tech Stack

| Technology | Usage |
|-----------|-------|

| Vanilla JavaScript | Interactivity and DOM manipulation |

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/tawchifulislam/My-Personal-Portfolio.git

# Navigate into the project
cd My-Personal-Portfolio

# Open in browser
open index.html
# Or just drag index.html into your browser
```

> No build step needed — it's pure HTML/CSS/JS!
